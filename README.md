# Django Ninja Login

Django Ninja Login is a simple Django project for implementing user authentication and login functionality using Django Ninja.

## Features

- User registration
- User login
- Password reset
- User profile management

## Requirements

- Python 3.x
- Django 3.x
- Django Ninja
